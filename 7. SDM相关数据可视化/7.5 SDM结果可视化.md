### 比较建模图层之间的差异

```R
## 注意这里尽量是相同空间序列的不同时间图层；
pa1 <- raster(ass )
pa1[] <- ifelse(ass[] >= 377, 1, 0)
pa2 <- raster(asg)
pa2[] <- ifelse(asg[] >= 377, 1, 0)
chp <- pa2 - pa1
plot(chp,col=c('red','gray','blue'))
```
